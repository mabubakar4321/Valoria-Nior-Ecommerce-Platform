# Valoria Nior Ecommerce Platform

Valoria Nior is a modern full-stack ecommerce platform built to deliver a seamless shopping experience for customers and a powerful management system for administrators.

The project consists of a dynamic storefront and a feature-rich admin dashboard designed for scalability, performance, and maintainability.

---

## 🚀 Overview

Valoria Nior provides:

- A responsive customer-facing ecommerce website
- A secure and robust admin dashboard
- Complete product and inventory management
- Order processing and tracking system
- User authentication and authorization
- Scalable architecture for future growth

---

## 🏗 Architecture

The platform follows a clean modular architecture:

- Frontend (Customer Website)
- Admin Dashboard
- Backend API
- Database Layer

Designed with separation of concerns and maintainable code structure.

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/valoria-nior-ecommerce.git
cd valoria-nior-ecommerce
```

### 2. Install Dependencies

Backend:
```bash
composer install
```

Frontend:
```bash
npm install
```

### 3. Configure Environment

Create a `.env` file and configure:

- Database credentials
- App URL
- API keys

### 4. Run Development Server

Backend:
```bash
php artisan serve
```

Frontend:
```bash
npm run dev
```
